# NAME

Dist::Zilla::App::Command::diffrelease - diff a build with a distribution from CPAN

# VERSION

Version 0.001

# SYNOPSIS

    $ dzil diffrelease

# DESCRIPTION

Diff a build with the latest distribution from CPAN.
